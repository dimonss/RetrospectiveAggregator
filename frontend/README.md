# Retrospective Aggregator — Frontend

Клиентская часть платформы для проведения ретроспектив.

## Требования

*(Укажите требования, например: Node.js >= 18, npm / yarn / pnpm)*

## Установка и запуск

1. Перейдите в директорию фронтенда:
   ```bash
   cd frontend
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
