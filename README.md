# Retrospective Aggregator (Платформа для ретроспектив)

Это платформа для проведения и агрегации результатов ретроспектив. Проект разделен на фронтенд и бэкенд части.

## Структура проекта

- [frontend/](./frontend/README.md) — Клиентская часть приложения
- [backend/](./backend/README.md) — Серверная часть приложения

Подробные инструкции по запуску и настройке каждого из компонентов находятся в соответствующих директориях:
- Инструкция по работе с фронтендом: [frontend/README.md](./frontend/README.md)
- Инструкция по работе с бэкендом: [backend/README.md](./backend/README.md)
